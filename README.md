# 💰 Finance Bot — инструкция по запуску

## Что делает бот
- Автоматически загружает транзакции из **Monobank** каждую неделю
- Принимает **CSV файлы** из Revolut, PayPal, Sparkasse
- Распределяет траты по категориям
- Записывает всё в вашу **Google таблицу**
- Присылает еженедельную сводку в Телеграм

## Команды бота
- `/start` — приветствие и список команд
- `/sync` — загрузить транзакции из Monobank прямо сейчас
- `/summary` — сводка расходов за текущий месяц
- `/weekly` — включить автоматическую еженедельную синхронизацию
- 📎 Отправить CSV файл — загрузить данные из Revolut/PayPal/Sparkasse

## Деплой на Railway (бесплатно)

### Шаг 1 — Создайте аккаунт
1. Зайдите на [railway.app](https://railway.app)
2. Войдите через GitHub

### Шаг 2 — Создайте проект
1. New Project → Deploy from GitHub repo
2. Загрузите файлы бота (или создайте репозиторий на GitHub)

### Шаг 3 — Настройте переменные окружения
В Railway → ваш проект → Variables → добавьте:

| Переменная | Значение |
|---|---|
| `TELEGRAM_TOKEN` | токен от @BotFather |
| `MONOBANK_TOKEN` | токен из приложения Mono |
| `SPREADSHEET_ID` | `1IZib9YydNs2vVPTu8Zdd_Vwpd3JphSDNTZKPzOs5LTA` |
| `GOOGLE_CREDENTIALS_JSON` | содержимое JSON файла (всё в одну строку) |

### Шаг 4 — Запустите
Railway автоматически запустит бота. Напишите `/start` вашему боту в Телеграм.

### Шаг 5 — Включите еженедельную синхронизацию
Напишите боту `/weekly` — он будет автоматически присылать сводку каждое воскресенье.

---

## Как загрузить GOOGLE_CREDENTIALS_JSON
Откройте JSON файл → скопируйте всё содержимое → вставьте в поле переменной в Railway.
Важно: всё должно быть в одну строку без переносов.

---

## Как скачать CSV из банков

**Revolut:** Приложение → профиль → Выписки → выбрать период → CSV

**PayPal:** [paypal.com](https://paypal.com) → Активность → Загрузить историю → CSV

**Sparkasse:** Онлайн-банк → Umsätze → Export → CSV (разделитель точка с запятой)

Скачанный файл просто отправьте боту в Телеграм — он сам разберёт формат!
